The professional certificate consists of nine courses. Projects and labs for each course are organized into corresponding directories:

1.  **What is Data Science?**
2.  **Tools for Data Science**
3.  **Data Science Methodology**
4.  **Python for Data Science, AI & Development**
    * *Includes Python basics, data structures, and foundational libraries like Pandas and NumPy.*
5.  **Databases and SQL for Data Science with Python**
    * *Focuses on relational databases, SQL queries, and interacting with databases from Python.*
6.  **Data Analysis with Python**
    * *Covers exploratory data analysis (EDA), data wrangling, and statistical analysis.*
7.  **Data Visualization with Python**
    * *Projects using Matplotlib, Seaborn, and Folium for various visualization tasks.*
8.  **Machine Learning with Python**
    * *Implementation of supervised (e.g., Regression, Classification) and unsupervised (e.g., Clustering) learning algorithms.*
9.  **Applied Data Science Capstone**
    * *The final capstone project involving a real-world problem (e.g., location data analysis, business case).*
